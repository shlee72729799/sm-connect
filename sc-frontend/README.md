# SM Connect - 커뮤니티 웹사이트

서강대학교 커뮤니티 웹사이트의 React 프론트엔드 프로젝트입니다.

## 주요 기능

- 📱 **메인 페이지**: 게시판 섹션과 인기글 표시
- 📄 **상세 페이지**: 게시글 상세 보기 및 카테고리 네비게이션
- 👤 **회원가입**: 이메일 인증을 포함한 회원 가입
- 🔐 **로그인**: 자동 로그인 기능 포함

## 기술 스택

- React 18
- React Router DOM
- Vite
- CSS3

## 설치 및 실행

### 의존성 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

개발 서버는 `http://localhost:3000`에서 실행됩니다.

### 빌드
```bash
npm run build
```

## 페이지 구조

### 1. 메인 페이지 (`/`)
- 게시판 섹션 그리드 레이아웃
- 오늘의 인기글, 자유게시판, 익명게시판 등
- 헤더 네비게이션 및 푸터

### 2. 상세 페이지 (`/detail`)
- 좌측 사이드바에 카테고리 목록
- 우측 게시글 상세 내용
- 카테고리별 게시글 수 표시

### 3. 회원가입 페이지 (`/register`)
- 아이디, 비밀번호, 닉네임 입력
- 서강대 이메일 인증 기능
- 양식 검증

### 4. 로그인 페이지 (`/login`)
- 간단한 로그인 폼
- 자동 로그인 체크박스
- 계정/비밀번호 찾기 링크

## 디자인 특징

- 파란색 그라데이션 배경
- 깔끔하고 현대적인 UI
- 반응형 디자인
- 흰색 카드 기반 레이아웃
